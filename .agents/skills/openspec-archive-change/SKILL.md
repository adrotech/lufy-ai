# OpenSpec Archive Change

Use when the user asks to archive a completed change.

1. Confirm tasks are complete and synced where required.
2. Confirm validation and delivery/sync gates from evidence.
3. Block archive when tasks remain incomplete or remote checks are missing.
4. Preserve a concise closure summary.
