# OpenSpec Apply Change

Use when an approved OpenSpec change is ready for implementation.

1. Read proposal, design, tasks, and affected spec deltas.
2. Implement only the current task/block or review slice.
3. Keep validation grouped unless a blocker requires diagnosis.
4. Do not mark work closed without validation and required delivery gates.
