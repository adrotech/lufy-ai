# OpenSpec Explore

Use for discovery before proposal or implementation.

1. Clarify problem, affected capabilities, risks, and candidate acceptance criteria.
2. Read relevant specs and files before recommending a path.
3. Do not implement changes.
4. Return concise next steps for propose, apply, verify, or no-op.
