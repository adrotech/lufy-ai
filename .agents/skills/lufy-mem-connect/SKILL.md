# Lufy Memory Connect

Use when memory notes need links, map updates, or relationship cleanup.

1. Search existing `.lufy/memory` notes before adding links.
2. Prefer relative Obsidian-safe links.
3. Do not invent notes or stronger evidence than repo files/commands.
4. Validate memory structure after edits.
