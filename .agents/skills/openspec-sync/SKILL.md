# OpenSpec Sync

Use when validated deltas need to update canonical specs.

1. Map each delta to `openspec/specs/<capability>/spec.md`.
2. Preserve requirement titles and scenarios accurately.
3. Validate synced specs after changes.
4. Do not archive unless archive preconditions are separately satisfied.
